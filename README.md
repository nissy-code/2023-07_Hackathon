## 構築方法
<p> プロジェクトを作成したいディレクトリまで移動.</p>
<p> 以下のコマンドを実行. </p>
<hr>

```git clone https://github.com/nissy-code/2023-07_Hackathon.git```

```cd virtual_namecard```

```npm install react react-dom react-router-dom```

```npm start```

```npm start``` で, 自分のPCでReactのページを確認できる.

### 使用技術
- React
- React-Router
- MaterialUI
- Firebase
